# home docs

This [home](https://home.bearcove.eu) site is deployed to <https://home.bearcove.eu>

To run locally, install home:

```bash
brew tap bearcove/tap
brew install home
```

Then, in the `docs/` directory serve:

```bash
home serve --open
```
